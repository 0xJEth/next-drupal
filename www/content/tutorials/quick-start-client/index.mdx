This tutorial will guide you through how to get started with Next.js for Drupal using the [`DrupalClient`](/docs/client).

You will learn how to setup Drupal and connect a Next.js site to display your content.

<Callout icon="⚠️">

The `DrupalClient` is available in `next-drupal ^1.3.0` as **experimental**.

</Callout>

Let's get started!
