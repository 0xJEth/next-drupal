## Generate keys

1. Visit _/admin/config/people/simple_oauth_
2. Click **Generate keys** to generate encryption keys for tokens
3. Fill in **Directory for the keys**.
4. Click **Generate**.

## Create Consumer

1. Visit _/admin/config/services/consumer/add_
2. Fill in the following values:

- **Label**: `Next.js site`
- **User**: `Select the user we created`
- **Secret**: `Your secret`
- **Scopes**: `Select the role we created`

3. Click **Save**

_Important: note the client id (uuid) and the secret. This is going to be used as environment variables for the Next.js site._
