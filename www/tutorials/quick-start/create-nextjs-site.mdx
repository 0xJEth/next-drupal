1. Visit _/admin/config/services/next_
2. Click **Add Next.js site**
3. Fill in the following values:

- **Label**: `Blog`
- **Base URL**: `http://localhost:3000`
- **Preview URL**: `http://localhost:3000/api/preview`
- **Preview Secret**: `preview-secret-here`

4. Click **Save**
