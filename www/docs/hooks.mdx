## useMenu

<Callout type="accent">

You need to install the [JSON:API Menu Items](https://www.google.com/search?q=drupal+jsonapi_menu_items) module to use `useMenu` and `getMenu`.

</Callout>

Returns menu items for a menu by name.

```ts
function useMenu(
  name: string
): {
  items: DrupalMenuLinkContent[]
  tree: DrupalMenuLinkContent[]
  error: unknown
  isLoading: boolean
}
```

- `items` is an array of menu items sorted by weight.
- `tree` is the hierarchical menu tree with parent and children.

#### Examples

```ts
const { tree, items } = useMenu("main")
```
